# Zenn tetunori articles

# Description🖊️
Articles for [tetunori](https://zenn.dev/tetunori).

# Usage🪄
## Make a new article
```bash
$ npx zenn new:article --slug YYYYMMDDNN-tetunori-article
```

## Preview
```bash
$ npx zenn preview
```
